# first-file

A tiny demo HTML page that displays a short "Levitating HTML" animation. This repository is a simple starting point for learning how static HTML, CSS, and GitHub Pages work.

## Preview
Open `index.html` in your browser or enable GitHub Pages for this repository to view it online.

## What changed
- The HTML has been moved to `index.html`.
- The page uses modern, accessible HTML5 and CSS (no deprecated `bgcolor` or `text` attributes).

## Files
- `index.html` — the demo page (HTML + CSS).
- `README.md` — this file.

## How to run locally
1. Clone the repository:
   git clone https://github.com/Swe010/first-file.git
2. Open `index.html` in your browser:
   - Double-click the file, or
   - Serve it with a simple static server, e.g. with Python:
     python3 -m http.server 8000
     Then visit http://localhost:8000 in your browser.

## Notes & accessibility
- Uses semantic tags and sets the document language.
- Uses CSS for color and layout to ensure better compatibility and accessibility.
- Color choices should be checked for sufficient contrast; swap colors if necessary.
